# PS1Back

Este es el backend del proyecto PS1Back desarrollado en JavaScript.

## Descripción

PS1Back es un proyecto que proporciona funcionalidades backend, escrito en JavaScript. Su propósito principal es gestionar la lógica y las operaciones del servidor para la aplicación asociada.

## Características principales

- 100% JavaScript
- Arquitectura modular y escalable
- Fácil integración con frontend
- Implementación de buenas prácticas de desarrollo

## Instalación

1. Clona el repositorio:

   ```bash
   git clone https://github.com/jose230203/PS1Back.git
   cd PS1Back
   ```

2. Instala las dependencias:

   ```bash
   npm install
   ```

## Uso

Para iniciar el servidor de desarrollo, ejecuta:

```bash
cd api
node index.js
```

Asegúrate de tener configurados los archivos de entorno necesarios y de que las dependencias están instaladas.

## Estructura del proyecto

- `src/` — Código fuente principal del backend
- `package.json` — Dependencias y scripts
